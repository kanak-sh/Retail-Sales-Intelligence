# Retail Sales Intelligence

## Overview

Retail Sales Intelligence is an end-to-end analytics project that transforms raw e-commerce sales data into actionable business insights using SQL, Python, and Power BI.

## Tech Stack

- Python
- SQL (MySQL/PostgreSQL)
- Power BI
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset

Olist Brazilian E-Commerce Dataset

## Project Goals

- Clean and preprocess raw sales data
- Design a star schema database
- Perform advanced SQL analysis
- Conduct exploratory data analysis using Python
- Build an interactive Power BI dashboard
- Generate business recommendations
